## Installation

> Blockchain part

```bash
# terminal 1
npx hardhat node

# terminal 2
npx hardhat compile
npx hardhat run scripts/deploy.js --network localhost
```
Copies one of the `Private Key` from the `terminal 1` and connect it to Metamask
to see the balance.

> Front-end part

```bash
npm install
npm run dev
```
